Team Page

Technical Description

The Team Page is structured using a flexible layout with CSS and HTML. It employs a div container to organize team members in a visually appealing grid. Each member's profile consists of an image, name, role, and hidden details that are revealed upon hovering. The hover effect is achieved through CSS transitions and transformations, creating a dynamic user experience. The position: relative and z-index properties ensure layered visual effects, while background: rgba(0, 0, 0, 0.6) is used to maintain readability against a semi-transparent background.

Accessibility

Inorder to enhance accessibility, semantic HTML elements such as h1, p, and ul are used to provide proper document structure. Alternative text (alt) is included for images, ensuring screen readers can interpret them. The hover-based interactions are supplemented with clear labels and text for users who may not rely on a mouse.

Feedback Form

Technical Description

The Feedback Form is built using HTML, CSS, and JavaScript to ensure an interactive and user-friendly experience. Input fields include text, email, tel, radio buttons, checkboxes, and a textarea. JavaScript validation is implemented to prevent invalid input, such as numbers in the name field and letters in the phone number field. CSS styles ensure a clean form layout, using border-radius for rounded edges and hover effects for interactivity.

Accessibility

The form elements include label tags associated with their respective inputs for better screen reader navigation. The form uses high-contrast colors and large buttons to improve usability. Error handling messages are displayed using alerts, which are accessible to keyboard users and screen readers.

Content Page

Technical Description

This page uses a structured layout to present information on deep-sea exploration. A flexbox-based .container is used to align content, ensuring responsive design. Headers (h1, h2) help organize topics, while p tags provide readable paragraph spacing. Images are wrapped in .image-container divs, using max-width: 50% and border-radius for aesthetic appeal. Bullet points (ul, li) highlight key information efficiently.

Accessibility

Proper heading structures (h1 to h2) improve screen reader navigation. Images include alt attributes for users who rely on text descriptions. The content is designed with a high-contrast background and readable font sizes to accommodate users with visual impairments.
