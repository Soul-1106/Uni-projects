This Python-based project management system enables efficient tracking, updating, and organization of projects for XYZ Company. The system allows users to add new projects, update project details, remove completed projects, and track overall project statistics. It also manages worker assignments and ensures projects do not exceed available workforce limits.

Key Features:

Project Management: Add, update, and remove projects.

Worker Allocation: Tracks available workers and prevents over-assignment.

Project Status Tracking: Categorizes projects as Ongoing, Completed, or On Hold.

Statistics Dashboard: Displays the number of ongoing, completed, and on-hold projects.

User-friendly Interface: Interactive menu-driven system for ease of use.

Built using Python, this system is designed for businesses looking to streamline project management while optimizing worker allocation.
